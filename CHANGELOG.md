# Changelog

## 0.1.0

- Initial release
- Implement checkPermissionStatus() - check current permission status

- Implement checkServiceStatus() - check if service is enabled
- Implement openAppSettings() - placeholder (not directly supported)
- Support common permissions: camera, storage, location, microphone, calendar, contacts, SMS, phone, bluetooth
- Example app with demo
